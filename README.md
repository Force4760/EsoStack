# EsoStack

A stack oriented eso-lang made in Go

## Tech Stack
* Languages: Golang
* Packages: 
    * Cobra: Command line interface
    * Flinch: Command line input
    * Testify: Improved Testing
* Others:
    * Make
    * Git
    * Markdown
    * Cookiecutter
